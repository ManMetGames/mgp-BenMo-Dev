# MGP_2526
 The assessment repo for 2526

Controls:
WASD - Basic Movement
Left Shift - Sprinting (Momentum Building)
Spacebar - Jumping / WallJumping
WallRunning - Jump towards a wall that has a "Wall" tag placed in it.
(The two walls in the middle of Lvl_FirstPerson)

MMUTube Assessment Video: https://mmutube.mmu.ac.uk/media/t/1_tfgniw8m

The Project can be downloaded via the GitHub Repo (https://github.com/ManMetGames/mgp-BenMo-Dev)
No extra software is required.
